## About This Project

ergodnc (Ergonomic Desk & Coffee) is an open source Laravel project that's being built live on the official Laravel YouTube Channel ([Check Here](https://www.youtube.com/watch?v=2RlbXNjhQkc&list=PLcjapmjyX17gCa-eo19wDXf3fD7H48NP5&ab_channel=Laravel)).

You can contribute by asking questions, helping others, reporting issues, or opening pull requests.
